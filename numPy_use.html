Let's take a look at the numPy arrays

Getting started

Step 1:

First step that we have to do in order to use numPy library is to install numPy library. In order to that go to your systems terminal and type the command : "pip install numpy"
pip is a package-management system written in python.

Step 2:

Second step is to import the numPy library in your jupyter notebook. We can import the numPy library using an import statement : import numPy"
After that run the code cell once so that we can use this library in our jupyter notebook. But if you don't want to use the whole name while using the numPy arrays what we can do is instead of writing "import numpy", we can write "import numpy as np". The word "as" is a keyword which literally says that import numpy as np,it means where ever we have to use "numpy" instead of that we can just use "np".

Step 3:

Hurray! You've intalled the library and you've imported it too know is the time to use our beloved numpPy library.
